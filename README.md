# Terminal-Project

This Porject aimed to re-create the "terminal" system as found on most operating systems. The system mimics a real terminal and suports the following commands:
1.) "exit" 
2.) "cd"
3.) "pwd"
4.) "ls"
5.) "ll"
6.) "touch"
7.) "mkdir"
8.) "rm"
9.) "mv"
10.) "tag"
11.) "tree"
12.) "help"
